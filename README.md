## üîÑ How Does the Sync Work?

sap-woocommerce-sync uses a queue system. This means it lines up all data changes and sends them one by one. This makes sure no information gets lost or mixed up.

The circuit breaker pattern helps prevent problems. If an error is detected, the sync pauses. This keeps your data safe until issues are fixed.
